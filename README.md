# ROCK PAPER SCISSORS GAME IN PYTHON 

The game is written in python and describes some of the commands and built in functions commonly used in Python. 
It also includes some Emojis for entertainment purpose. 
In addition to the Rock-Paper-Scissors game, there are some lessons on the fundementals of Python. 
This repository will be improved with different applications and further lessons related to fundemental and in depth Python applications.
