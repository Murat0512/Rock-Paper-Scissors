# ROCK PAPER SCISSORS GAME IN PYTHON

The game is written in python and describes some of the commands and built in functions commonly used in Python.
It also includes some Emojis for entertainment purpose.
In addition to the Rock-Paper-Scissors game, there are some lessons on the fundementals of Python.
This repository will be improved with different applications and further lessons related to fundemental and in depth Python applications.

This repository was created by following the steps below.

VSCode Terminal
git
git init
Open the folders on VSCode
define your credentials
Check your credentials git config --list
Check your branch ( git branch)
git add .
git commit
Go to GitHub and create a new repository,
Obtain the link for that repository
Back to VSCode
Type git remote add origin + URL address for the new repository(THe link copy and paste)
git push -u origin master ( u means push everything upstream) (Make sure the branch name is matching. This can either be Main or master)
This will push all the files upstream into GitHub repository.

This section was editited on readme branch to practice pull and push request. Pull request was completed directly using git pull command as it was directly modified online on GitHub and changes were saved. Since the job is not shared by anyone
It was taken into local repository.

We will now push this change back into GitHub.
This time the files will be merged and pull request will be completed.
